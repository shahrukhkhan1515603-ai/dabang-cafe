# 🔴 Dabbang Cafe - Web App

Welcome to the Dabbang Cafe website! A modern, responsive web application showcasing the cafe's brand, timings, and services.

## 🌟 Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Eye-catching red and gold theme matching the Dabbang Cafe branding
- **Smooth Animations**: Engaging scroll animations and hover effects
- **Hero Section**: Prominent branding with animated elements
- **Timings Display**: Highlights the new extended operating hours (12 PM - 2 AM)
- **Menu Section**: Showcase of different food categories
- **Contact Information**: Easy access to location and contact details
- **Mobile Optimized**: Touch-friendly interface for all devices

## 🔧 Customization

### Update Cafe Information

Edit the following in `index.html`:

```html
<!-- Contact Section -->
<div class="contact-item">
    <h3>📍 Location</h3>
    <p>Dabbang Cafe<br>Your City, Your Street</p>
</div>

<div class="contact-item">
    <h3>📞 Phone</h3>
    <p>+91-XXXX-XXXX-XX</p>
</div>
```

### Change Colors

Update CSS variables in `styles.css`:

```css
:root {
    --primary-red: #e63946;
    --secondary-red: #d62828;
    --gold: #ffc107;
    /* ... more colors ... */
}
```

### Add Your Logo

Replace the emoji logo with your image:

```html
<div class="logo">
    <img src="path/to/your/logo.png" alt="Dabbang Cafe Logo">
</div>
```
